# ✨ 核心功能亮点

#### 🚀 三种创建方式
- ⌨️ **快捷键** `Alt+Ctrl+C` — 选中文字一键批注
- 👆 **滑动批注** — 手机/桌面拖选连续创建，沉浸式标注
- 🔘 **工具栏按钮** — 编辑器浮动工具栏一键添加

#### 🎨 正文下划线 · 35 种组合
- 7 种颜色 × 5 种线型，纯 DOM 渲染，**不修改正文 markdown**
- 支持一键更新当前文档全部下划线样式

#### 📋 侧边栏管理
- 卡片展示原文预览 + 批注内容，三种排序方式
- 点击卡片跳转正文高亮或文末批注块
- 孤儿批注自动检测，提示清理

#### 🏷️ 全局标签批注
- 五种标签：📌 置顶 · ⭐ 重要 · 📖 阅读 · ✔️ 待办 · ❓ 存疑
- **跨文档聚合** — 所有标签批注集中展示，标签筛选搜索，一键跳转源文档
- 标签筛选自动记忆，折叠栏整条可点击

#### 📍 三种插入位置
- 文档底部 · 今日日记 · 目标文档
- 各自独立头部模板（支持 `{年}{月}{日}{来源块引用}` 等占位符）
- 手动选择模式：每次创建弹出三选一

#### ✏️ Protyle 弹窗编辑
- 完整富文本编辑，`Ctrl+Enter` 保存
- 清洁模式（隐藏标记）/ 标号模式，三种底层列表结构
- 自定义复制格式，三套独立时间格式

#### 📱 手机端深度优化
- 滑动批注 RAF 节流 + `deferAttrs` 伺服加速，松手即弹窗
- 弹窗 / 选择器透明遮罩，点击外面只关闭不弹输入法

#### 🌓 暗色模式
自动检测思源主题，侧边栏 Apple 风格适配

#### 🛡️ 数据零污染
所有元数据存块属性，正文 kramdown 干净，导出同步不受影响

### 创建批注

| 方式 | 操作 |
|------|------|
| 快捷键 | 选中文字 → `Alt+Ctrl+C`（Windows）/ `⌥⌃C`（macOS） |
| 工具栏 | 选中文字 → 浮动工具栏批注按钮 |
| 滑动模式 | 侧栏开启「滑动快速批注」→ 正文拖选 |

开启手动选择位置时，创建瞬间弹出三选一：← → 方向键切换 + Enter 确认。

### 查看与编辑
- **侧栏**：桌面端右下角 Dock / 移动端顶栏图标
- **点击下划线**：打开编辑弹窗
- **侧栏卡片**：原文区 → 跳转高亮；内容区 → 跳转文末批注块

### 弹窗操作

| 操作 | 说明 |
|------|------|
| `Ctrl+Enter` / 保存按钮 | 保存并关闭 |
| 复制 | 按自定义格式复制到剪贴板 |
| 跳转 | 跨文档定位批注位置 |
| 删除 | 删除批注（含确认） |
| `Esc` | 关闭弹窗 |

### 标签管理
侧栏卡片点标签图标 → 弹出标签菜单 → 选择标签。标签批注在侧栏顶部独立区域展示。

### 设置入口
侧栏右上角 ⚙ 按钮。

# 📦 安装

思源笔记 → **设置 → 集市 → 插件** → 搜索「鲸鱼快速批注」→ 安装。

或从 [Releases](https://github.com/HaoCeans/comment/releases) 下载 `package.zip`，解压至 `data/plugins/siyuan-comment/`。

> 环境要求：思源笔记 ≥ 3.0.0，桌面端与移动端均支持。
